GLOSARIO:

*InnerHTML: Es una propiedad de los elementos HTML que representa el contenido HTML dentro de ese elemento. Permite obtener o modificar el contenido HTML de un elemento.

*Push: Es un método de los arrays en JavaScript que se utiliza para añadir uno o más elementos al final de un array y devuelve la nueva longitud del array.

*Math.floor: Es una función en JavaScript que devuelve el número entero más grande menor o igual que un número dado.

*Includes: Es un método de los arrays en JavaScript que determina si un array incluye un determinado elemento, devolviendo true o false según corresponda.

*AddEventListener: Es un método en JavaScript que se utiliza para adjuntar un evento a un elemento HTML, especificando el tipo de evento y la función que se ejecutará cuando ocurra ese evento.

*Sort: Es un método de los arrays en JavaScript que se utiliza para ordenar los elementos de un array. Por defecto, ordena los elementos como strings y los ordena alfabéticamente.

*ForEach: Es un método de los arrays en JavaScript que ejecuta una función una vez por cada elemento del array.

*Onchange: Es un atributo de los elementos HTML que se utiliza para especificar una función que se ejecutará cuando el valor del elemento cambie, como en el caso de los inputs o select.
